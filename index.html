Combination Sequence Twitter+ Google+ Github

Purpose

Given 2 sequences of numbers with the same length, this simple program will print out all possible combinations from the two from each index.

For example:
Sequence A: [1][3]; Sequence B: [2][4]
Results: [1][3], [1][4], [2][3], [2][4]
